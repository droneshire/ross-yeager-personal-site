# Ross Yeager - Personal Site

Minimal personal website built with plain HTML/CSS.

## Local Development
Open `index.html` in a browser.

## Deployment
Designed for GitHub Pages. Push to `droneshire.github.io` repository.

## Customize
- Update projects in `index.html`
- Modify colors/fonts in the `<style>` section
- Add custom domain via `CNAME` file

Built with zero dependencies. Fast, simple, accessible.
